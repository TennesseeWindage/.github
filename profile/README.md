# TennesseeWindage

**Precision systems for the modern field operator.**

We build open and restricted-access tools spanning ballistics, targeting, communication, and tactical awareness — from embedded firmware to server infrastructure.

---

## Public Repositories

### [OpenReticle](https://github.com/TennesseeWindage/OpenReticle)
An open standard and reference implementation for communicating reticle data (holdovers, corrections, reticle selection, and zero information) from a smart system — such as a rangefinder or targeting computer — to a scope or optic over **I²C** and **Bluetooth Serial**. Designed to be hardware-agnostic and easy to integrate into existing systems.

### [TN-TAK-Server](https://github.com/TennesseeWindage/TN-TAK-Server)
A Docker-wrapped, opinionated deployment of the **TAK Server** (Team Awareness Kit) with streamlined configuration, hardened defaults, and scripted setup for rapid field or organizational deployment.
